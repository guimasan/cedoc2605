# Canvas Handoff - CEACA CEDOC

Pacote de entrega com três protótipos estáticos para o projeto CEACA CEDOC.

## Conteúdo

- [Hub comparativo](../index.html)
- [Protótipo 1](../prototipo-1/index.html)
- [Protótipo 2](../prototipo-2/index.html)
- [Protótipo 3](../prototipo-3/index.html)
- [Manifest](./prototypes-manifest.json)

## Uso sugerido

1. Abra o hub para comparar as três direções.
2. Use o manifest para localizar rápido a pasta de cada versão.
3. Repassa a pasta `canvas-handoff` para o time de design como ponto de partida.

## Observações

- Cada protótipo é independente e possui seu próprio `index.html` e `style.css`.
- As 3 versões compartilham conteúdo-base, mas diferem em tom visual e composição.
- A pasta foi pensada para facilitar exportação e adaptação posterior em canvas.
